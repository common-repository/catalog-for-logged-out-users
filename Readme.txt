== Description ==

Catalog for Logged Out Users is a robust WordPress plugin that can help you to convert woo commerce store into a catalog. You can show any digital and analog product without showing the price and cart button for logged out user. There are lots of customization option will be there.  Here are our some bold features"- 

<h3>Include features</h3>

*   Hide price from shop page
*   Hide price from single product
*   Hide cart button from shop page
*   Hide cart button from single product
*   Add custom text with custom url
*   Fully customizable
*   Compitable with Latest version
*   Full Responsive
*   Clean Design
*   Translation Ready

N.B: If you want more options in this plugin, please [contact with us](https://itech-softsolutions.com/ "iTech Softsolutions")

== Installation ==

= Minimum Requirements =

* WordPress 4.9 or greater
* PHP version 7.0 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 7.4 or greater
* MySQL version 5.6 or greater
* WP Memory limit of 64 MB or greater (128 MB or higher is preferred)
* You need to install contact form 7

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Pages > Add New
4. Press the 'Edit with Elementor' button.
5. Now you have the Elements Buddy from the left panel which yu can drag onto the content area.

= Setup =
1. You need to go WooCommerce setting page here you will find one tab "Catalog for Logged Out Users". Here you will find all the options to configure the plugin.

<strong>Before installing this plugin, you must have to install "WooCommerce"</strong>
